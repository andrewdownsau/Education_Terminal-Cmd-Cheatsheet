  • cd
    • Change Directory (switch the current working directory)
    • .(current) ..(parent) ~(home) /(root)

    • Touch (Creates new file with defined name and extension)
    • "." prefix for hidden
  • mkdir
    • Make Directory (Creates new folder/directory with name)
    • "." prefix for hidden -p(create parents as needed)
  • rm
    • Remove (Deletes file or directory/contents permanently)
    • -r(recursive) -f(forced)
  • mv
    • Move (Moves file contents to new target file (name/location changed))
  • cp
    • Copy (Copy file or directory with name/location)
    • -r(recursive for directories)
  • cat
    • Catenate (Combine file contents and print output to terminal)

    • Echo text (Print text after echo on terminal output)
